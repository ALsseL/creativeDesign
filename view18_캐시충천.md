# 인터페이스 페이지 세부요소
  ## 배경이미지 (element-00)
	- 왼쪽에 주유기는 보충하는 이미지가 있음
	- 게임 전채형상에 따라서 밑에 잡초가 무성하는 이미지가 있음
  ## 구매정보

| 게임화폐 (element-01) | 추가 보너스 | 결산금액(원) (element-02) |
| --------------------- | ----------- | ------------------------- |
| 60                    | 0           | 1100                      |
| 180                   | 10          | 3300                      |
| 600                   | 60          | 11000                     |
| 1500                  | 300         | 27500                     |
| 2950                  | 900         | 55000                     |
| 5900                  | 2200        | 110000                    |
  ## 상품 추천 표시
	- 빨간색 | 인기 상품
	- 노란색 | 특혜 상품
